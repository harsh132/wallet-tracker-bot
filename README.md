# wallet-tracker-bot
Telegram notification for incoming transactions on Polygon.

Try It Out At:

```https://t.me/WebThreeBot```

## Installation

```bash
npm install
```
Create a `.env` file with the following contents:
```
WS_RPC=<polygon-websocket-rpc>
TELEGRAM_BOT_TOKEN=<your-telegram-bot-token>
```
Then run:
```bash
node tracker.js
```
